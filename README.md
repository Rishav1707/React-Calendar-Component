## Run Locally

Clone the project

```bash
  git clone https://github.com/your_username/React-Calendar-Component.git
```

Go to the project directory

```bash
  cd Calendar-Component
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
